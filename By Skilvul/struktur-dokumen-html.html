Apa itu HTML?

HTML adalah bahasa komputer yang digunakan untuk membuat kerangka atau struktur untuk Web pages (halaman website) di internet. HTML adalah singkatan dari Hyper Text Markup Language.

Bagaimana HTML Bekerja di Web Browser?

Web browser seperti Chrome, Firefox, Edge, Safari, atau Opera akan membaca dokumen HTML. Dokumen HTML yang berisi tag-tag HTML akan memberitahu browser bagaimana cara menampilkan sebuah konten. Selain HTML, ada juga bahasa CSS (Cascading Style Sheet) dan JavaScript yang akan kita pelajari untuk membuat sebuah tampilan website yang interaktif.

Secara fungsi, ketiga bahasa tersebut bisa dianalogikan seperti ini:

Fungsi HTML adalah sebagai 'kerangka', yang memberi struktur pada website.

Fungsi CSS sebagai 'baju' , yang memberi warna dan layout pada website.

Fungsi JavaScript, sebagai element yang membuat website menjadi interaktif.
